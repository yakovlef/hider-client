# Hider Client

Невидимое окно для одностороннего чата. Не захватывается при screen share.

## Запуск

```bash
npm install
npm start
```

## Использование

1. Ввести адрес сервера (IP:порт)
2. Выбрать режим: **Редактор** или **Читатель**
3. Нажать «Подключиться»

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Cmd+\` | Скрыть / показать окно |
| `Alt+стрелки` | Двигать окно |
| `Cmd+Shift+E` | Экстренное закрытие |

## Сборка

```bash
npm run make
```
